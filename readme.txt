== Description ==
A simple SEO Plugin which generates for Pages, Posts, Categories:

- og:type
- og:title
- og:description
- og:image
- twitter:title
- twitter:description
- twitter:image
- Meta Robots Tag: index, Follow, NoFollow, NoIndex

= The plugin is new multilingual =
Available in English & German
  
  
= Plugin Recommendation =
If you need a Sitemap for your WordPress - please install <a href="https://wordpress.org/plugins/simple-xml-sitemap-generator/" target="_blank">Simple XML Sitemap Generator</a>

If you need a Google Analytics Plugin for your WordPress - please install <a href="https://de.wordpress.org/plugins/quick-google-analytics/" target="_blank">Quick Google Analytics</a>
